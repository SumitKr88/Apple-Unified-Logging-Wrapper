# Apple-Unified-Logging-Wrapper
Wrapper class for implementing Apple Unified Logging mechanism 

Usage Examples :

`Log.debugLog("Test %@", log: .network, "value")`

`Log.defaultLog("Test")`

`Log.defaultLog("Test %@", log: .network, "value")`
